## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочкой (*) Например:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пукнты просто пронумеровать. Например:
1. Первый пукнт
2. Второй пункт 
3. Третий пункт
